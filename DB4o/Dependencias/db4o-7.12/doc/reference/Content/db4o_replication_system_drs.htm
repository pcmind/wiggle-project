db4o Replication System

The db4o Replication System provides functionality to periodically synchronize databases that work disconnected from each other. For example mobile devices synchronizing to a central server or desktop application.

The dRS can replicate between any number of db4o-databases. It can replicate unidirectional or bidirectional.

Utilizing Hibernate dRS can also replicate between db4o and a RDBMS.

Before you start, please make sure that you have downloaded the latest dRS distribution from the download areaof the db4o developer website.
